WordPress Installer
===================

## Requirements

* [WordPress Requirements][3]
  * PHP 5.2.4 or greater
  * MySQL 5.0 or greater
  * Apache HTTP Server
* [cURL][4] must be installed
* [allow_url_fopen][5] must be enabled
